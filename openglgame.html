OPEN-GL GAME

This project was done for the Object-oriented programming + Rendering exam. This code is entirely written in C++ and uses OpenGL library. This game consists in a timed race. The goal is to complete 4 laps as quickly as possible.
To develop this game, I created some 3D models with Blender and used some free ones found online (like environment). I defined a class for the kart that contains informations about dimension and position. The movement of the kart is determinated with uniformly accelerated motion. To check collision, I created a bounding box and invisible walls. For each recursion, I checked the intesection with a simple calculation. I implemented a shader with a purely diffusive brdf with bright colors to make the atmosphere of the cartoon. I also added some reflective elements (like coins and the engine) and a shadow mapping. I used a Sky-Box as background.
